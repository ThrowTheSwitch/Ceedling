# `wondrous_forest`

:material-forest-outline: An imagined forest monitoring system project
illustrating the use of [Partials](../../testing-guide/partials/index.md)
in a test suite.

## Exporting

```shell
ceedling example wondrous_forest [destination]
```

## About

`wondrous_forest` simulates a forest environmental monitoring system. It is
designed specifically to demonstrate Ceedling's
[Partials](../../testing-guide/partials/index.md) feature, which allows test
files to be split across multiple source files and assembled at build time.

After exporting, run `ceedling test:all` from the project root to execute the
full test suite.
