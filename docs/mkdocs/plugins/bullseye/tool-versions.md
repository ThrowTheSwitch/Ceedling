# Supported Tool Versions

## Bullseye Coverage

This plugin has been verified against Bullseye Coverage 9.25.7 (Linux x64).
Its `covc` compiler/linker wrapping, `covsrc`/`covfn` console reporting,
`covhtml` HTML reporting, and `covselect` report exclusions all rely on
command-line conventions that have been stable across Bullseye's release
history, so earlier and later Bullseye versions are also likely to work,
but have not been specifically verified.

If you use this plugin successfully (or run into trouble) with a different
Bullseye version or platform, that's useful information for improving this
documentation over time.
