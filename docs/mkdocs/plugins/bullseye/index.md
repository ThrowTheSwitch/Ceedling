# Bullseye

The `bullseye` plugin integrates the commercial [Bullseye Coverage][bullseye]
tool with test builds. It provides function and condition/decision coverage
summaries to the console by default and can optionally produce a full
interactive HTML coverage report.

!!! warning "Requires a commercial license"
    Unlike Ceedling's `gcov` plugin, Bullseye Coverage is not free, open
    source software. You need a working Bullseye installation and license to
    use this plugin. See [Licensing][licensing].
